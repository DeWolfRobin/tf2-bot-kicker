# tf2-botkicker by boyonk

REQUIREMENTS
- Python 3 (or later)

SETUP
1. Startup Steam and go to your library
2. Right-click on Team Fortress 2
3. Click "Properties"
4. Go to "General"
5. Under "Launch Options" add "-condebug"
6. Change config.properties accordingly
7. You're good to go!
